# 알고리즘

어떤 문제들은 인간보다 컴퓨터가 하는게 더 쉬울 때가 있다.

어떤 존재=객체

모든 것은 객체다

파이썬에서 가장 유명한 말

## 할당연산자

**=**

3의 객체의 주소에  a객체가 포인팅 한다.

ex) a = 3 3이라는 존재가 생기고 a가 3이라는 존재에 포인팅한다.

a는 상자에 3이라고 레이블링함

초기설정을 할당이라는 수단으로 함.

```python
name = 'mulcam'
age = 3 
is_adult = False
```

리스트는 대괄호안에 정보들을 넣는 것

리스트 활용

```python
mulcam_info = ['mulcam',3,False]
```

리스트 안에 ,는 트레일링 콤마

딕셔너리 활용

```python
mulcam_info {
        'name':'mulcam',
        'age':3,
        'is_adult' : False

}
```

~에 ~

여러가지 자료형들이 있는데 그것들을 구조화시키는 방법에 대해서 알아봄. 

### 활용하기 좋게 **구조화** 할 수 있는가

리스트

여러명의 데이터를 넣을 수 있다.

딕셔너리

여러가지 정보를 넣을 수 있다.

컴퓨터는 인덱스를 0부터 시작

문제에 맞게 구조화를 항상 신경쓸 것!

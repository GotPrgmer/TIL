# 프로그래밍이란?

## 프로그래밍이란?

- 프로그램을 만드는 행위
    - 프로그램이란?
    - 특정 작업(컴퓨터가 수행해야 하는)을 수행하기 위한 일련의(순서있는) 명령어의 모음
    - 컴퓨터에게 일을 시키기 위해 프로그램을 만드는 행위
    - 컴퓨터에게 일을 시키는 행위

# 프로그래밍 언어란?

- 생각을 전달하기 위해 사용하는 체계
- 문법적으로 맞고 이해할 수 잇는 말의 집합
- 컴퓨터는 0과1(기계어)로 소통! 전기가 들어오면 1 들어오지 않으면 0
- 기계어의 대안으로 사람이 사용한 프로그래밍언어가 필요
- 소스코드 필요
    - 프로그래밍으로 작성된 프로그램
- 번역기 필요
    - 소스코드를 컴퓨터가 이해할 수 있는 기계어로 번역
    - 파이썬의 경우 인터프리터 사용

# 파이썬이란?

하이레벨 언어이기 때문에 사람친화적, 기계어로 번역하는 데 시간이 많이 걸림

## 배워야 하는 이유

- 알고리즘 코딩 테스트에 유리
- 구현 코딩 테스트에 유리
    - 유용한 라이브러리 중 최소한만 사용해 프로그램을 개발 가능
    - 실행시간이 매우 중요한 문제 유형이 아닌 입사 코딩테스트에서는 파이썬이 유리
- 가장 인기가 많은 언어
- 배우기 쉽다
    - 프로그래밍 사고에 집중 할 수 있음
    - 문법이 간결하고 유연함
    - 변수에 별도의 타입저장이 필요없음
- 인터프리터 언어
    - 소스코드를 기계어로 변환할 때 통역하듯이 1줄씩 변환
- 객체 지향 프로그래밍
    - 객체지향 언어이므로, 모든것이 객체로 구현되어 있음

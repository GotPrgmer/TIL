# .gitignore

---

.gitignore에  명시된 파일은 절대로 올라가지 않는다.(git의 추적을 피하겠습니다.)

bin폴더는 github에 올라가지 않는다.

.gitkeep?이건 뭐지(구글링 해봐야함)

github는 빈폴더를 푸싱시키지않는데 gitkeep파일에 해당 폴더 명을 쓰면 푸싱시킨다.

내가 한번이라도 사진찍힌 애면 효력범위를 벗어남

처음부터 git ignore 설정해야함

git ignore의 파일을 자동적으로 생성해주는 사이트가 있음.

(언어별로 생성 시킬 수 있다.)

[`https://www.toptal.com/developers/gitignore/`](https://www.toptal.com/developers/gitignore/)
